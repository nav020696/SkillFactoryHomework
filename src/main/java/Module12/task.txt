Напишите класс Person. В нём определите следующие поля:
String name;
String surname;
int age;
String post (должность).

В классе создайте констуктор со всеми полями. Сделайте все поля приватными. Подумайте, какие поля следует сделать final
и которые не должны изменяться.
Создайте к полям геттеры и сеттеры (для которых нужно, не для всех).
Переопределите методы equals() и hashcode() и toString();

Создайте класс Main. В классе создайте экземпляр класса Map <Person, Integer>, в котором ключ - экземпляр класса Person,
а значение сумма дохода.
Удалите из Map позиции, где сумма ждохода меньше 15000.
Выведите на экран оставшиеся позиции.
