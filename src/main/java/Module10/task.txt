Задание:
В файле найти самую длинную строку и строку с самым большим количеством слов.
Если встречается несколько строк, имеющих самую большую длину и количество слов, то записать последнюю.
Записать эти строки в новый файл в виде:
Самая длинная строка: + строка. Её длина: + значение длины строки;
Максимальное количество слов в строке: + строка. Количество слов: + количество.
