Запишите в этот же файл на новую строку сумму этих чисел. Формат вывода в файл: Сумма чисел равна + значение суммы.

Создайте своё исключение, которое вы будете вызывать, если в файле для чтения в строке расположено не два числа.

Обработайте все возможные исключения, которые могут возникнуть в процессе работы программы.
